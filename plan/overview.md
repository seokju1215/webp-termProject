# 서비스 개요

## 핵심 아이디어

QA 상황에서 기획자와 개발자가 효율적으로 소통하고, AI 자동화까지 지원하는 QA 협업 플랫폼.

기획자는 개발자가 등록한 프로젝트에 접속해 QA를 진행하고, 화면의 특정 DOM 요소에 직접 댓글을 달아 피드백을 남긴다. 개발자는 MCP 서버를 연결해 AI가 해당 피드백을 자동으로 참고해 수정 작업을 수행할 수 있다.

## 주요 특징

- **DOM 기반 댓글**: 기획자가 화면의 특정 UI 요소를 직접 선택해 댓글을 달 수 있음 (Agentation 외부 API 활용)
- **테스트 케이스**: 댓글과 독립적으로 QA 진행 상황을 추적하는 체크리스트 단위
- **MCP 서버 연동**: 댓글과 DOM 위치 정보가 개발자의 AI 도구로 자동 전달
- **팀/워크스페이스**: 팀 단위로 프로젝트와 멤버를 관리
- **통합 UI**: 기획자/개발자 기능이 사이드바에서 구분되며 누구나 모든 기능에 접근 가능

## 서비스 플로우

```
[개발자]
프로젝트 등록 (이름, URL, 대표 이미지) → Agentation 연동 설정 → QA 요청

[기획자]
팀 프로젝트 접속 → 테스트 케이스 생성 → iframe으로 프로젝트 확인
→ Agentation으로 DOM 요소 선택 → 댓글 작성 → 테스트 케이스 완료/실패 처리

[AI 자동화]
개발자 MCP 서버 연결 → 댓글 + DOM 위치 정보 수신
→ AI가 자동으로 코드 수정
```
